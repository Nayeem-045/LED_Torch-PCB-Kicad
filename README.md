# LED_Torch-PCB-Kicad
## Objectives:

* Design a simple battery-powered LED torch
* Practice schematic capture in KiCad
* Perform ERC and DRC checks

## Tools Used

* KiCad (EDA Tool)
* PCB Design Rules
* ERC (Electrical Rule Check)
* DRC (Design Rule Check)

## Features

* Battery powered operation
* Compact PCB footprint
* High-brightness LED drive
* Simple and low-cost design
* Suitable for beginners in PCB design

## Project Files

* Schematic file (.kicad_sch)
* PCB layout (.kicad_pcb)
* Project file (.kicad_pro)
